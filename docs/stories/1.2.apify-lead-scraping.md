## Story
**As a** usuário,
**I want** acionar uma busca de prospects no Instagram ou LinkedIn via API,
**so that** eu possa capturar leads do meu nicho (clínicas odontológicas/estéticas) automaticamente sem fazer manualmente.

## Acceptance Criteria
1. Endpoint `POST /api/scrape` aceita body `{ platform: "instagram"|"linkedin", keyword: string, limit: number }`
2. A rota integra com `apify-client` usando `APIFY_TOKEN` do `.env.local`
3. O scraping retorna ao menos os campos: `name`, `profileUrl`, `platform`, `bio`, `followersCount`
4. Erros da Apify API são capturados e retornam resposta `{ error: string }` com status 500
5. A rota funciona corretamente testada via `curl` ou Postman

## Dev Notes

### ApifyService — Implementação
[Source: docs/architecture.md#apifyservice]
```typescript
// src/lib/apify-service.ts
import { ApifyClient } from "apify-client";
import type { ScrapeRequest } from "@/types/scrape";
import type { Lead } from "@/types/lead";

const ACTORS: Record<string, string> = {
  instagram: "apify/instagram-profile-scraper",
  linkedin:  "apify/linkedin-profile-scraper",
};

const client = new ApifyClient({ token: process.env.APIFY_TOKEN });

export async function scrapeProspects(
  request: ScrapeRequest
): Promise<Partial<Lead>[]> {
  const actorId = ACTORS[request.platform];
  const run = await client.actor(actorId).call({
    searchTerms: [request.keyword],
    maxItems: request.limit,
  });
  const { items } = await client.dataset(run.defaultDatasetId).listItems();

  return items.map((item: any) => ({
    name:           item.fullName ?? item.name ?? "",
    profileUrl:     item.url ?? item.profileUrl ?? "",
    platform:       request.platform,
    bio:            item.biography ?? item.summary ?? "",
    followersCount: item.followersCount ?? item.connectionsCount ?? 0,
  }));
}
```

### Route Handler — Padrão
[Source: docs/architecture.md#backend-architecture]
```typescript
// src/app/api/scrape/route.ts
import { NextRequest, NextResponse } from "next/server";
import { scrapeProspects } from "@/lib/apify-service";

export async function POST(request: NextRequest) {
  try {
    const body = await request.json();
    // validar body...
    const leads = await scrapeProspects(body);
    return NextResponse.json({ leads, count: leads.length });
  } catch (error) {
    console.error("[POST /api/scrape]", error);
    return NextResponse.json({ error: "Falha no scraping" }, { status: 500 });
  }
}
```
