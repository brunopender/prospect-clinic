## Story
**As a** usuário,
**I want** que o sistema gere automaticamente uma mensagem personalizada para cada lead via Gemini API,
**so that** eu economize tempo na elaboração de abordagens e tenha mensagens relevantes para cada prospect.

## Acceptance Criteria
1. Endpoint `POST /api/leads/[id]/generate-message` integra com `@google/generative-ai` usando `GEMINI_API_KEY`
2. O prompt enviado ao Gemini inclui: nome, bio, plataforma e seguidores do lead
3. A mensagem gerada é salva no campo `message` do lead no JSON via `leadsRepository.update()`
4. Mensagem gerada em até 10 segundos (timeout configurável via `GEMINI_TIMEOUT_MS`)
5. Erros da Gemini API retornam `{ error: string }` com status 500

## Dev Notes

### GeminiService — Implementação
[Source: docs/architecture.md#geminiservice]
```typescript
// src/lib/gemini-service.ts
import { GoogleGenerativeAI } from "@google/generative-ai";
import type { Lead } from "@/types/lead";

const genAI = new GoogleGenerativeAI(process.env.GEMINI_API_KEY!);
const model = genAI.getGenerativeModel({ model: "gemini-1.5-flash" });

const TIMEOUT_MS = parseInt(process.env.GEMINI_TIMEOUT_MS ?? "10000", 10);

function buildPrompt(lead: Lead): string {
  return `Você é um especialista em prospecção B2B para clínicas de saúde e estética no Brasil.

Escreva uma mensagem de primeiro contato para o seguinte prospect no ${lead.platform}:
- Nome: ${lead.name}
- Bio: ${lead.bio}
- Seguidores: ${lead.followersCount}

A mensagem deve:
- Ter no máximo 150 palavras
- Ser pessoal e mencionar algo específico do perfil
- Apresentar brevemente soluções de IA para clínicas
- Ter um call-to-action para uma conversa de 15 minutos
- Soar natural, não robótica

Escreva apenas a mensagem, sem explicações.`;
}

export async function generateMessage(lead: Lead): Promise<string> {
  const timeoutPromise = new Promise<never>((_, reject) =>
    setTimeout(() => reject(new Error("Gemini timeout")), TIMEOUT_MS)
  );
  const generatePromise = model
    .generateContent(buildPrompt(lead))
    .then(r => r.response.text());

  return Promise.race([generatePromise, timeoutPromise]);
}
```

### Route Handler — Padrão
[Source: docs/architecture.md#backend-architecture]
```typescript
// src/app/api/leads/[id]/generate-message/route.ts
import { NextRequest, NextResponse } from "next/server";
import { leadsRepository } from "@/lib/leads-repository";
import { generateMessage } from "@/lib/gemini-service";

export async function POST(
  _request: NextRequest,
  { params }: { params: { id: string } }
) {
  try {
    const lead = await leadsRepository.getById(params.id);
    if (!lead) {
      return NextResponse.json({ error: "Lead não encontrado" }, { status: 404 });
    }
    const message = await generateMessage(lead);
    await leadsRepository.update(params.id, { message });
    return NextResponse.json({ message });
  } catch (error) {
    console.error("[POST /api/leads/[id]/generate-message]", error);
    return NextResponse.json({ error: "Falha ao gerar mensagem" }, { status: 500 });
  }
}
```
